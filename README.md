### 功能特性

#### 登录页面 (login.html)
- 现代化的登录界面设计，参照 Whalet 官方风格
- 渐变背景和浮动动画效果
- 表单验证（邮箱格式、密码长度）
- 密码可见性切换
- 记住我功能
- 加载动画和错误提示
- 完全响应式设计，适配移动端

#### 主应用 (index.html)
- 登录状态检查和保护
- 用户信息显示
- 安全退出功能
- 完整的合作伙伴门户功能

### 使用方法

1. 直接访问 `index.html` 会自动重定向到登录页面
2. 在登录页面输入演示账号信息
3. 登录成功后自动跳转到主应用
4. 点击右上角的"Logout"按钮可以退出登录

### 技术实现

- 使用 sessionStorage 存储登录状态
- 基于 JavaScript 的前端路由和状态管理
- 响应式设计，支持桌面端和移动端
- 表单验证和用户体验优化

### 安全特性

- 登录状态验证
- 自动重定向未认证用户
- 安全的退出机制
- 表单输入验证
